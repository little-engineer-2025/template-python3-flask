## Getting started

### Once tasks

- Install dependencies:
```sh
sudo dnf install -y python3-poetry
make deps
```

### Frequent tasks

- Install dependencies: `make deps`
- Code at `my_service`
- Unit test at `tests`
- Integration tests at `tests/integration`
- Apply format, run linter, and run unit tests by: `make all`
- Get help by: `make help`
